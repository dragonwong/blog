--- layout: article title: select 在 ie6 下使用 innerHTML 填充 bug tags: diary ---

html:

<select id="select"></select>

js:

var select = document.getElementById('select'),
	options = [];

for(var j = 1; j <= 6; j++){
    if(j == 3){
        options.push( '' );
    }else{
        options.push( '' );
    }
};

select.innerHTML = options.join('');

在 ie6 下，第一个 option 标签会被吞掉，如下图：

针对泯灭人性的 ie6，使用 outerHTML ，或者对 select 的父元素用 innerHTML
