--- layout: article title: command in javascript console tags: js console ---

selector

It is amazing that $() can be used in console. $() is like querySelector() while $$() is like querySelectorAll(). $0 and $1 will return the latest selected element by debugger.

time

console.time() and console.timeEnd() provide an easy to time.

profile

console.profile() and console.profileEnd(). Better in firebug.

log

Not only console.log(), but also console.info() and console.warn().

count

console.count() will count the time the function is called.
