I started to use Atom these days. I try to write snippets and I find the document is not so specific. For example, I tried to override Atom's native snippets but failed and I do not why. See detail: How to override Atom's native snippets?

To my surprise, Atom's snippet takes a strategy like CSS selector instead of the suffix of file name to match files. We can find 'classes' of a file by the following steps: Confirm the language of the file, for example it is html, then find atom/language-html/grammars/html.cson, in the first line you can see 'scopeName': 'text.html.basic', so html files's 'classes' is 'text', 'html' and 'basic'.
