snippet

I tried to override Atom's native snippets but failed. See this issue: How to override Atom's native snippets?

To my surprise, Atom's snippet takes a strategy like CSS selector instead of the suffix of file name to match files. We can find 'classes' of a file by the following steps: Confirm the language of the file, suppose it is html, then find atom/language-html/grammars/html.cson, in the first line you can see 'scopeName': 'text.html.basic', so html files' 'classes' are 'text', 'html' and 'basic'.

The above is all based on my thought, if it is wrong I will not pay you .
