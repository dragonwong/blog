--- layout: article title: simple operations of mongodb tags: mongodb ---

Report the name of the current database:

db

Display the list of databases:

show dbs

Create or switch to a new database:

use mydb

Delete the current database:

db.dropDatabase()

Create a database and insert a document:

j = { name : "mongo" }
db.myCollection.insert( j )

Display the list of collections:

show collections

Delete a database:

db.myCollection.drop()
