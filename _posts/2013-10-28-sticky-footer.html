--- layout: article title: Sticky Footer ---

以前也遇到过，要让footer一直吸附于页面底部，这个需求还是挺多的。今天来整理一下。

主要还是学的人家的：《CSS Sticky Footer》。自己也写了一个demo，小优化下了，看代码。

html：

<body>  
    <div class="main">
        <header>
            ...
        </header>

        <div class="body">
            ...
        </div>

    </div>

    <footer class="foot">
        ...
    </footer>
</body>

css（只给出关键的样式）：

    /* main css */  
    html{
        height: 100%;
    }
    body{
        height: 100%;
    }
        .main{
            min-height: 100%;
            height: auto !important;
            height: 100%;
        }
            .body{
                padding-bottom: 40px;
            }
        footer{
            margin-top: -40px;
            height: 40px;
            line-height: 40px;
        }

自己做了一个优化，去除了<div class=".push">，于是控制<footer>和<div class="main">间隙的任务就落到<div class="main">中最后一个元素（即<div class="body">）身上。给<div class="body">添加样式padding-bottom即可，距离至少是要是<footer>的高。
