--- layout: post title: install mongodb in windows tags: mongodb ---

offical guide: http://docs.mongodb.org/manual/tutorial/install-mongodb-on-windows/

How to run?

open a command prompt, and run mongod.exe:

d:\mongodb\bin\mongod.exe --dbpath d:\mongodb\data

use --dbpath to specify an alternate path. But I don't know how to make it as default.

Then, open another command prompt and run mongo.exe:

d:\mongodb\bin\mongo.exe

now, you can run mongo command here.
