Note several points in fe I met these days. (ps: I have been in Qunar as internship since this Monday.)

1. In css, '' is not need in (). For example:

    background: url(dancheng.png) no-repeat;

2. In js, Setting the operation + in the front of code could be used to convert String to Number, but will not change the value. For example:

var a = '1',
    b = +a;

//  a -> '1'
//  b -> 1

This method is more efficient than parseFloat().

3. When we need to scroll the page, use document.documentElement.scrollTop, do not use document.body.scrollTop

4. Declarative value and funtion could not be deleted. For example:

var a = 10;
delete a;
alert(a); // 10

function func() {}
delete func;
alert(func); // func code

For more info, visit 《JavaScript中delete操作符不能删除的对象》
