This blog is written by jekyll which use a templating engine called Liquid. Like almost other templating engines, Liquid use output tag looking like: {{output}}.

Now, how to display liquid code? It is obvious that I can not write {{“{{output“}}}} directly. The solution is:

Put the code between {%raw%} and {%endraw%}

For example, if you want to output:

{%raw%}{{post.title}}{%endraw%}

You can write like this:

{%raw%}{%raw%}{{post.title}}{%endraw%}
