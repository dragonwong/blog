How to open cmd with the current address?

Simply type in 'cmd' in the address bar and it'll open the command prompt with the path to your current folder already set.

How to create a file which is not allowed by windows?

For example, we need to create a file named '.fekitrc', so we come into cmd and type in 'notepad .fekitrc'.

How to keep a program running which I started from cmd if I logout from cmd?

Use command &.
